# 데이터 이전 가이드

> 🚧 **이 문서는 준비 중입니다.**
>
> RisuAI(웹/Tauri/로컬 Node)에서 PocketRisu로 이전하는 상세 절차가 여기에 정리될 예정입니다.

당분간은 세 가지 방법이 가능합니다:

1. **로컬 백업(`.bin`)** — RisuAI의 설정 > 백업에서 내보낸 뒤, PocketRisu의 설정 > 데이터 이전에서 불러오기.
2. **Save 폴더 zip 업로드** — RisuAI의 `save/` 폴더를 zip으로 압축한 뒤, 설정 > 데이터 이전에서 업로드.
3. **Save 폴더 직접 복사** — PocketRisu 정지 → `save/` 폴더 덮어쓰기 → 재시작하면 자동 마이그레이션 진행. 대용량 데이터에 권장.

가이드가 완성되기 전까지 [기존 README](../../README.legacy.md)에서 전체 절차를 참고하세요.
