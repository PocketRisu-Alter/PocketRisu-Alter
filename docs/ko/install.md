# 설치 가이드

> 🚧 **이 문서는 준비 중입니다.**
>
> 포터블 배포, Docker, 서버 스크립트, 소스 빌드에 대한 상세 안내가 여기에 정리될 예정입니다.

당분간은:

- [Releases 페이지](https://github.com/PocketRisu/PocketRisu/releases)에서 포터블 패키지 다운로드
- Docker로 설치:
  ```bash
  curl -L https://raw.githubusercontent.com/PocketRisu/PocketRisu/main/docker-compose.yml -o docker-compose.yml
  docker compose up -d
  ```
- 또는 가이드가 완성되기 전까지 [기존 README](../../README.legacy.md)에서 전체 설치 절차를 참고하세요.
