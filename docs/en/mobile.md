# Mobile Access Guide

> 🚧 **This document is being prepared.**
>
> Detailed instructions for Quick Tunnel and Tailscale will go here.

For now:

- **Quick Tunnel** — In PocketRisu, go to Settings → Remote Access → "Open Remote Access" to generate a temporary URL + QR code. Easiest, no setup needed. URL changes on server restart.
- **Tailscale** — Install Tailscale on PC and phone with the same account, then run `tailscale serve --bg http://localhost:6001` on the PC. Persistent URL across restarts.

See the [legacy README](../../README.legacy.md) (Korean) for full step-by-step instructions until this guide is complete.
