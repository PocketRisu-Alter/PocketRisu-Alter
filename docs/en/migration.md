# Migration Guide

> 🚧 **This document is being prepared.**
>
> Detailed migration steps from RisuAI (web / Tauri / local Node) to PocketRisu will go here.

For now, three options are available:

1. **Local backup (`.bin`)** — Export from RisuAI's Settings → Backup, then import in PocketRisu's Settings → Data Migration.
2. **Save folder zip** — Compress your RisuAI `save/` folder and upload via Settings → Data Migration.
3. **Save folder direct copy** — Stop PocketRisu, overwrite the `save/` folder, then restart for automatic migration. Best for large datasets.

See the [legacy README](../../README.legacy.md) (Korean) for full step-by-step instructions until this guide is complete.
